# 家庭菜单计划与购物清单生成工具 - 产品需求文档

## 1. 产品概述

家庭菜单计划与购物清单生成工具是一款帮助家庭规划每周饮食、自动生成购物清单的Web应用。用户可以为一周7天规划早中晚餐，系统根据菜单自动汇总所需食材并按类别整理，支持家庭成员协作编辑，提高家庭饮食规划效率。

- **核心价值**：解决"今天吃什么"的日常难题，减少食材浪费，提升购物效率
- **目标用户**：家庭主妇/主夫、注重健康饮食的家庭、需要规划饮食的减脂人群
- **产品定位**：轻量级、高颜值的家庭饮食规划工具

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 家庭成员 | 本地存储/协作码加入 | 编辑菜单、勾选食材、使用模板 |
| 家庭管理员 | 创建家庭组 | 管理成员、管理模板、导出清单 |

### 2.2 功能模块

1. **菜单计划页面**：周视图日历、每日三餐编辑、菜品库选择
2. **购物清单页面**：食材分类展示、勾选已有食材、数量调整
3. **模板管理页面**：保存/加载菜单模板、模板分类、模板编辑
4. **导出打印页面**：PDF导出、打印预览、清单格式选择
5. **家庭协作页面**：成员管理、协作码分享、实时同步

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 菜单计划页 | 周视图导航 | 7天横向排布，可切换周，显示每天三餐概览 |
| 菜单计划页 | 餐次编辑卡片 | 点击某天某餐弹出编辑面板，可添加/删除菜品，设置份量 |
| 菜单计划页 | 菜品库侧边栏 | 常用菜品快速选择，支持搜索和分类筛选 |
| 购物清单页 | 分类食材列表 | 按蔬菜、肉类、调料、主食等分类展示，显示总用量 |
| 购物清单页 | 已有食材勾选 | 勾选框标记家中已有食材，实时从待购清单中移除 |
| 购物清单页 | 数量调整 | 支持增减食材用量，自动计算总价（可选） |
| 模板管理页 | 模板卡片列表 | 展示已保存的模板，支持预览、应用、删除 |

| 家庭协作页 | 协作码管理 | 生成/重置协作码，设置有效期 |

## 3. 核心流程

### 3.1 菜单规划流程

用户进入菜单计划页 → 选择某天某餐 → 从菜品库选择或手动添加菜品 → 系统自动计算食材用量 → 切换到购物清单页查看汇总

```mermaid
flowchart LR
    A["打开菜单计划页"] --> B["选择日期和餐次"]
    B --> C["添加菜品"]
    C --> D["选择菜品库/手动输入"]
    D --> E["保存菜单"]
    E --> F["系统自动计算食材"]
    F --> G["生成购物清单"]
```

### 3.2 购物清单使用流程

用户进入购物清单页 → 查看分类食材 → 勾选家中已有食材 → 调整数量 → 导出/打印清单

```mermaid
flowchart LR
    A["打开购物清单页"] --> B["按分类浏览食材"]
    B --> C["勾选已有食材"]
    C --> D["调整食材数量"]
    D --> E["确认待购清单"]
    E --> F["导出PDF/打印"]
```

### 3.3 模板使用流程

用户浏览模板库 → 选择模板 → 应用到当前周 → 可微调修改 → 保存为新模板

```mermaid
flowchart LR
    A["打开模板管理页"] --> B["浏览模板列表"]
    B --> C["选择模板预览"]
    C --> D["应用到当前周"]
    D --> E["调整菜单"]
    E --> F["保存为新模板"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **整体风格**：温暖有机、清新自然的居家风格，营造温馨的厨房氛围
- **主色调**：暖绿色（#2D5A27）- 代表新鲜、健康、自然
- **辅助色**：米白色（#FAF7F2）背景、橙红色（#E07A5F）强调色、暖黄色（#F2CC8F）点缀
- **按钮风格**：圆润边角，轻微阴影，悬停时有上浮效果
- **字体**：标题使用"ZCOOL XiaoWei"（文艺衬线体），正文使用"Noto Sans SC"（现代无衬线）
- **布局风格**：卡片式布局，柔和阴影，大量留白，呼吸感强
- **图标风格**：线性手绘风格图标，配合食物emoji增加趣味性

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 菜单计划页 | 顶部导航 | Logo、导航标签、家庭成员头像、协作状态指示 |
| 菜单计划页 | 周视图头部 | 周切换按钮、当前周日期范围、模板快速选择下拉 |

### 4.3 响应式设计

- **设计原则**：Desktop-first，移动端自适应
- **桌面端**：≥1024px，三栏布局或双栏布局，充分利用空间
- **平板端**：768-1024px，自适应列数，侧边栏可收起
- **手机端**：<768px，单列布局，底部导航，弹窗改为底部抽屉
- **触控优化**：按钮最小点击区域48x48px，列表项增大间距

### 4.4 动效与交互

- 页面切换：平滑淡入淡出，滑动过渡
- 卡片交互：悬停微上浮+阴影加深，点击缩放反馈
- 勾选动画：打勾时有弹性动画，列表项平滑过渡到已完成状态
- 食材添加：从菜品库拖入餐次卡片时的吸附动画
- 加载状态：食物图标旋转动画，骨架屏占位
